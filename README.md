# git_vue

## 项目说明
鉴于本地过多git项目，自己的和公司的容易混杂，在项目过多的情况下，路径、提交步骤等问题会让我的脑细胞出现急剧死亡的状态，因此开发这个项目。

## 目前功能
1. 根据本地路径自动识别 git地址以及 git分支，可展示、编辑、删除
2. 对某一个项目进行 pull 和 push 操作
3. 对所有项目进行 pull 和 push 操作
4. 查看 pull 和 push 操作执行结果
